# Blog Express ✍️

A simple blog platform built with React and Appwrite.

## Tech Stack ⚙️

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Appwrite (BaaS)
- **Authentication:** Google OAuth
- **State Management:** Redux Toolkit

## Features ✨

- **Authentication:** Google OAuth login/signup
- **Blog Management:** Create, read, update, delete posts
- **Rich Editor:** TinyMCE for post content
- **Image Upload:** Feature images for posts
- **Dark Mode:** Light/dark theme toggle
- **Responsive:** Mobile-friendly design

## Getting Started 🚀

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development: `npm run dev`
4. Open `http://localhost:5173`
